## How to Set Up

1. **Paste the plugin** into your dashboard
2. **Works immediately** in demo mode (orange dot = demo)
3. **For live data:**
   - Go to [finnhub.io](https://finnhub.io) → sign up free
   - Copy your API key
   - Edit the widget config → set `apiKey: "your_key_here"`
4. Green dot = live data 🟢

## Customization via `defaultConfig`

| Option | Default | Description |
|--------|---------|-------------|
| `symbols` | `["AAPL","MSFT",…]` | Array of ticker symbols |
| `apiKey` | `""` | Your Finnhub free API key |
| `refreshInterval` | `30` | Seconds between auto-refresh |
| `currency` | `"USD"` | Display currency label |

You can add any symbol Finnhub supports — stocks, ETFs (e.g. `"SPY"`, `"QQQ"`), even crypto like `"BINANCE:BTCUSDT"`.

### What I fixed

1. **`fetch()` → `api.fetch()`** — The original used the browser's native `fetch()` directly, which gets blocked by CORS when calling external APIs. `api.fetch()` routes through the dashboard's built-in proxy, bypassing that restriction.

2. **`document.getElementById()` → `container.querySelector()`** — Scoped all DOM lookups to the widget's container instead of the whole document, which is safer if multiple widgets are on the same page.
